# Graphic Maker

Backend de la aplicacion, donde se realizan validaciones de JWT, tiempo de sesion y usuarios registrados

Construida usando node.js

## Instalacion

Usar npm [yarn](https://yarnpkg.com/) para construir la aplicacion

```bash
yarn start
```
